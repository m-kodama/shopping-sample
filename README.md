フードデリバリーアプリの簡易サンプル（学習目的）
- UI構築
- ダークモード対応
- 画面遷移
- 状態管理
- MVVMアーキテクチャー

  - カートへ商品を追加する機能はなし

## 起動方法

- `flutter pub get`
- `flutter run`
